Elemento di catalogo: DeleteByPK

Descrizione

Operazione di cancellazione di un singolo record, identificato tramite chiave primaria

Caratteristiche (features)

cod.nomedescrizione
DELPK001 delete record by PK

Il record da cancellare viene selezionato tramite primary key (il cui valore deve essere fornito in input al metodo java corrispondente). Nel caso in cui la cancellazione violi uno o più vincoli di integrità referenziale il metodo corrispondente solleverà un'eccezione.
