Class "datagen :: QueryDef"

Defined in package : datagen

Definizione della query da eseguire. Utilizzando questo tipo di modellazione è possibile modellare interamente la query senza limitazioni sulla struttura dei campi in output, delle tabelle coinvolte, dei criteri di join. L'utilizzo di questa modalità inibisce la possibilità/necessità di scrivere codice java per implementare la query.

Reference datagen :: SelectClause selectClause

Determina l'elenco delle colonne che costituiscono il risultato della query. Le colonne possono essere di due tipi:

  • colonne appartenenti ad una qualsiasi delle tabelle coinvolte (ovvero quelle referenziate nella FromClause
  • colonne calcolate, ottenute ad esempio applicando funzioni di aggregazione a colonne effettive (es. somma), oppure applicando altri tipi di funzioni ad una o più colonne effettive (es. concatenzaione)
Il DTO associato al CustomFinder che utlizza questo tipo di costrutto conterrà i campi necessari a contenenre adeguatamente i risultati della query e non sarà il DTO classico associato al DAO.
