Elemento di catalogo: FindByPrimaryKey

Descrizione

Mediante questa operazione di ricerca è possibile estrarre un singolo record della tabella associata al DataAccessObject, effettuando una ricerca tramite chiave primaria.

Caratteristiche (features)

cod.nomedescrizione
FNDBYPK001 select by primary key

Il risultato dell'operazione FindByPK è un unico record la cui chiave primaria corrisponde a quella fornita in input. Se non viene trovata una corrispondenza alla chiave fornita il risultato è null.

FNDBYPK002 read main table columns

Il risultato della operazione contiene tutte le colonne della tabella principale associata al DataAccessObject, a meno che non venga utilizzata la funzione di LookupResolver (v.)

FNDBYPK003 read resolved lookup

A fronte della presenza nel DataAccessObject di un LookupResolver, il risultato della operazione contiene il DTO di lookup invece che le colonne corrispondenti alla ForeignKey. Il nome della property è derivato dall'attributo name del LookupResolver.

FNDBYPK004 cachable

Se il DataAccessObject lo prevede, l'operazione di FindByPK utilizza la cache
