Elemento di catalogo: CustomDeleter

Descrizione

Operazione di eliminazione dati che permette di cancellare fisicamente dal DB uno o più record. La logica di selezione è altamente personalizzabile tramite la scrittura di codice java.

Caratteristiche (features)

cod.nomedescrizione
DELCUS001 custom filter class

Per permettere la personalizzazione della logica di selezione dei record da eliminare è possibile specificare il full qualified name della classe che verrà utilizzata come parametro di input per contenere i criteri di selezione (filter class). E' possibile specificare il nome completo della classe oppure utilizzare gli shortcut ${dto} e ${dto[]) per indicare rispettivamente la classe del DTO associato al DAO e un array di tale classe. E' compito dello sviluppatore scriver eil codice java che sfrutta i valori contenuti in questo oggetto per effettuare la selezione dei record da cancellare.
