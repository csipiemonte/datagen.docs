Elemento di catalogo: Inserter

Descrizione

Rappresenta l'operazione che permette di inserire una singola riga (record) in una tabella

Caratteristiche (features)

cod.nomedescrizione
INSERT001 insert main table columns

L'Inserter permette l'inserimento nella tabella associata al DataAccessObject di un nuovo record. Le colonne del nuovo record saranno popolate con i valori del DTO in input al metodo (v. feature DAO001 - DataAccessObject). Nel caso in cui il DataAccessObject possieda dei LookupResolver per popolare i campi delle foreign-key corrispondenti verrà visitata la struttura del DTO di secondo livello (v. feature DAO002 - DataAccessObject).

INSERT002 sequence PK generator (single field)

Nel caso in cui la tabella preveda una primary key di tipo numerico (una sola colonna!) il cui valore debba essere popolato progressivamente mediante il supporto di una SEQUENCE, questo è possibile mediante il costrutto SequencePKGenerator. E' possibile impostare il nome della sequence da utilizzare. Il valore assegnato dal sistema al momento dell'inserimento popolerà il corrispondente campo del DTO che il metodo di inserimento restituisce in output al termine dell' operazione.
