Elemento di catalogo: CustomUpdater

Descrizione

Operazione di aggiornamento dati che permette di aggiornare tutte le colonne della tabella relativamente ad uno o più record. La logica di selezione e di attribuzione dei valori è altamente personalizzabile tramite la scrittura di codice java.

Caratteristiche (features)

cod.nomedescrizione
UPDCUS001 update main table columns

L'UpdateRow permette l'aggiornamento dei valori delle colonne di un record contenuto nella tabella associata al DataAccessObject. I record oggetto di aggiornamento sono selezionati con logica custom scritt ain linguaggio java all'interno del metodo di update.

UPDCUS002 custom filter class

Per permettere la personalizzazione della logica di selezione dei record da modificare è possibile specificare il full qualified name della classe che verrà utilizzata come parametro di input per contenere i criteri di selezione (filter class). E' possibile specificare il nome completo della classe oppure utilizzare gli shortcut ${dto} e ${dto[]) per indicare rispettivamente la classe del DTO associato al DAO e un array di tale classe. E' compito dello sviluppatore scriver eil codice java che sfrutta i valori contenuti in questo oggetto per effettuare la selezione dei record da modificare.

UPDCUS003 custom value class

Per permettere la personalizzazione della logica di attribuzione dei nuovi valori ai record da modificare è possibile specificare il full qualified name della classe che verrà utilizzata come parametro di input per contenere tali valori (value class). E' possibile specificare il nome completo della classe oppure utilizzare gli shortcut ${dto} e ${dto[]) per indicare rispettivamente la classe del DTO associato al DAO e un array di tale classe. E' compito dello sviluppatore scriver eil codice java che sfrutta i valori contenuti in questo oggetto per effettuare l'impostazione dei valori nei record da modificare.
