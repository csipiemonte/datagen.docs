Elemento di catalogo: UpdateRow

Descrizione

Operazione di aggiornamento dati che permette di aggiornare tutte le colonne della tabella relativamente ad un singolo record identitficato tramite primary key

Caratteristiche (features)

cod.nomedescrizione
UPDROW001 update main table columns

L'UpdateRow permette l'aggiornamento dei valori delle colonne di un record contenuto nella tabella associata al DataAccessObject. Il record oggetto di aggiornamento è quello identificato dalla primary key fornita in input al metodo di update. Le colonne del record saranno popolate con i valori del DTO in input al metodo (v. feature DAO001 - DataAccessObject), compresi i valori nulli. Nel caso in cui il DataAccessObject possieda dei LookupResolver per popolare i campi delle foreign-key corrispondenti verrà visitata la struttura del DTO di secondo livello (v. feature DAO002 - DataAccessObject).
