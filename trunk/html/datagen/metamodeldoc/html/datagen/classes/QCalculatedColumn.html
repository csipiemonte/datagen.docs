Attribute ecore :: EString freeTextExpression

Espressione che coinvolge le colonne referenziate nella property referencedColumns. L'espressione deve essere scritta nel dialetto SQL del DBMS target e può contenere segnaposto nella forma $i dove i rappresenta la posizione a partire da 1 nell'elenco delle colonne referenziate. L'espressione non può contenere una funzione di aggregazione.
